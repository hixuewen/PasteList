# Assets 资源文件说明

## 微信二维码

请在此目录下放置您的微信二维码图片，并重命名为：

```
wechat-qr-code.png
```

### 图片要求
- **格式**: PNG 或 JPG
- **尺寸**: 建议 200x200 像素或以上
- **清晰度**: 确保二维码清晰可扫描
- **文件大小**: 建议 500KB 以内

### 如何添加二维码
1. 将您的微信二维码图片复制到此 `assets/` 目录
2. 重命名为 `wechat-qr-code.png`
3. 确保 README.md 中的引用路径正确

### 注意事项
- 二维码图片将显示在项目 README.md 的联系作者部分
- 请确保微信二维码能够正常扫描和添加好友
- 建议在二维码旁添加"请备注 GitHub-PasteList"的提示文字（可通过图片编辑软件添加）

如果需要帮助生成微信二维码，可以：
1. 在微信中进入"我" → "服务" → "收付款"
2. 点击"向银行或手机号转账"下方的"二维码收款"
3. 截图保存二维码图片
